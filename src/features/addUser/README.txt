Задача: 
    Створити модальне вікно з формою для додавання нового юзера.

Декомпозиція задачі:
 - Верстка компоненту.
 - Створення слайсу для модального вікна, створення стейту isModalOpen із булевим значенням, по дефолту false.
 - написання функції closeModal , яка присвоює стейту isModalOpen значення false при натисканні на фон модального вікна або на кнопку Cancel завдяки useDispatch().
 - створення дочірнього компоненту з формою для додавання юзера
 - Підключення необхідних API для отримання списків країн, департаментів та статусів
 - Відображення лоадера, якщо дані з API ще не завантажились
 - Відображення повідомлення про помилки, якщо виникла проблема з API
 - Створення верстки форми, що включає поля для вводу повного імені, вибору департаменту, країни та статусу
 - Встановлення початкових значень для полів форми
 - Оновлення стану компоненту при виборі значення у випадаючих списках
 - Логіка увімкнення та вимкнення кнопки збереження в залежності від заповненості полів
 - Обробка події натискання на кнопку "Додати" — створення нового користувача, присвоєння йому унікального ID
 - Відправка даних нового користувача на сервер через запит до API
 - Закриття модального вікна після успішного додавання користувача
 - Обробка кнопки "Скасувати" — закриття модального вікна без додавання користувача
