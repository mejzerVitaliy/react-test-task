Задача:
    Створити сторінку для відображення списку користувачів з можливістю фільтрації та видалення.

Декомпозиція задачі:
 - Компоненти:
    UsersTable — основна таблиця користувачів.
    UsersList — рендеринг списку користувачів.
    UsersArray — рядки таблиці з даними користувачів.
 - Redux slice:
    Створити userSlice — зберігання користувачів та операції.
 - Фільтрація:
    Додати стейт для фільтрів.
    Компоненти для вибору фільтрів (відділ, країна, статус).
 - Видалення:
    Функція видалення користувачів.
    Підтвердження видалення.
 - Стан відображення:
    Лоадер для завантаження.
    Повідомлення про помилки.
