Задача: 
    Створити сторінку для редагування аккаунтів юзерів.

Декомпозиція задачі: 
- верстка сторінки
- написання запитів к АПІ завдяки РТК Квері та підтягування даних на сторінку
- створення дочірнього компоненту с налаштуваннями юзера, предаємо пропсами в цей компонент дані, отримані з АПІ
- верстка дочірнього компоненту
- задання дефолтного стейту для селекторів форми та для їх значень
- підтягування даних у селектор юзерів
- автоматичне підтягування даних юзерів до селекторів після вибору юзера
- зміна стейту при зміні значення селектору, а також зміна стейту для кнопки Undo
- зміна стейту оновленого юзера при кожній зміні селекторів
- написання логіки кнопки Undo, а саме скидання всіх стейтів до дефолтних
- написання логіки кнопки Save. Передаємо айді та оновленого юзера, а потім асинхронно шукаємо на сервері юзера за таким айді, і замінюємо його властивості на нові.
- декомпозування та оптимізація коду.
